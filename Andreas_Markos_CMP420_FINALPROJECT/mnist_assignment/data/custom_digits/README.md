# A2 - Νέα Χειρόγραφα Ψηφία

Η τελική εργασία απαιτεί **τουλάχιστον 20 νέα χειρόγραφα ψηφία που δημιούργησε ο φοιτητής**.

Βάλτε εδώ PNG/JPG/BMP εικόνες, μία εικόνα ανά ψηφίο.

Παραδείγματα filenames:

```text
0_01.png
0_02.png
1_01.jpg
7_03.png
9_02.png
```

Το πρώτο τμήμα του filename πριν από `_` είναι το πραγματικό label.

Μετά τρέξτε:

```powershell
python -m src.run_experiments
```

Το πρόγραμμα χρησιμοποιεί το ήδη εκπαιδευμένο μοντέλο του Α1 χωρίς επανεκπαίδευση και δημιουργεί:
- predictions CSV,
- classification report,
- confusion matrix,
- gallery των εικόνων με true/predicted labels.

Για την τελική υποβολή, μην χρησιμοποιείτε synthetic test digits αντί για δικά σας χειρόγραφα χωρίς σαφή ακαδημαϊκή δήλωση.
